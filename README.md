# Reverso 🚀🦾

**Reverso** es tu centro de mando para conexiones SSH y túneles. Olvídate de la terminal y las configuraciones complejas; gestiona todos tus servidores y redirecciones de puertos desde una interfaz visual moderna y ultra-rápida.

## 🌟 Características Principales

*   **Gestión Visual de Conexiones:** Guarda tus servidores SSH y conéctate con un solo clic.
*   **Túneles Simplificados (Port Forwarding):** Crea y activa túneles locales en segundos. Ideal para bases de datos (PostgreSQL, MySQL, etc.) o servicios internos.
*   **Dashboard Inteligente:** Una vista global de todos tus túneles activos, latencia promedio y servidores conectados.
*   **Organización por Grupos:** Clasifica tus servidores en carpetas personalizables con iconos y colores vibrantes.
*   **Buscador Inteligente:** Encuentra cualquier conexión por nombre, IP o usuario al instante.
*   **Control de Latencia:** Monitoriza la respuesta de tus servidores en tiempo real para evitar sorpresas.
*   **Terminal Integrada:** Abre múltiples pestañas de terminal directamente en cada conexión sin salir de la app.

## 📥 ¿Cómo funciona?

1.  **Añade una Conexión:** Introduce los datos de tu servidor (Host, Usuario, Puerto).
2.  **Crea un Túnel:** En la pestaña "Mappings", añade los puertos que quieras redirigir.
3.  **Conecta:** Pulsa el botón Play. La app se encargará de levantar la conexión SSH y todos los túneles automáticamente.
4.  **Monitorea:** Usa el "Active Tunnels" Dashboard para ver qué puertos tienes ocupados en tu ordenador local.

## 🛠️ Requisitos

*   Tener acceso SSH al servidor remoto.
*   Node.js (si lo ejecutas en modo desarrollo).
*   ¡Ganas de dejar de pelearte con los comandos `ssh -L`!
